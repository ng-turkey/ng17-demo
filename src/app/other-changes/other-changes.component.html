Signals Are now stable!

Signals API is now marked as stable.

effect() and RxJS interoperability functions (toSignal, toObservable) are still marked as developer preview. mutate has been dropped
  • Previously: books.mutate(books => books.push(newBook))
  • Now: books.update(books => [...books, newBook])

Http Improvements

Fetch backend has been promoted to stable.

When SSR enabled, it is now possible to customize transfer cache using withHttpTransferCacheOptions(options)
  • filter: a function to filter the requests that should be cached.
  • includeHeaders: the list of headers to include (empty by default)
  • includePostRequests: cache POST requests (only GET and HEAD requests are cached by default)
Example: 
        bootstrapApplication(AppComponent, {{ '{' }}
          providers: [provideHttpClient({{ '{' }}
            withHttpTransferCacheOptions({{ '{' }} includePostRequests: true {{ '}' }})
          {{ '}' }})
        {{ '}' }});
Lazy Load Animations Package

It is now possible to lazy-load the animations package.

        bootstrapApplication(AppComponent, {{ '{' }}

        {{ '}' }});
